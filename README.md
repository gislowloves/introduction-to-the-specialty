**The Programmer's Matryoshka Doll**

Лабораторная работа по дисциплине «Технологии программирования». Это многоуровневая система автоматической генерации кода, реализованная по принципу матрешки: каждый уровень системы порождает следующий на другом языке программирования.


**Суть проекта**

Реализация цепочки воспроизводства исполняемого кода из 4-х этапов. Конечная цель — работающая консольная игра «Камень, ножницы, бумага» с ботом-читером.


**Цепочка генерации:**

- Python (step3.py) — Генерирует Bash-скрипт.
- Bash (step2.sh) — Генерирует исходный код на C++.
- C++ (generator.cpp) — Создает финальный исполняемый Bash-скрипт.
- Bash (rsp.sh) — Финальная игра.


**Технологии**

- Docker: Изоляция среды и автоматизация сборки.
- GCC: Компиляция C++ кода внутри контейнера.
- Python 3: Начальная автоматизация.
- Bash: Логика игры.


**Инструкции по запуску**

Весь процесс генерации происходит внутри Docker-контейнера. Вам не нужно устанавливать компиляторы или интерпретаторы локально.

1. Сборка системы
Команда запускает «цепную реакцию» генерации всех уровней кода:

```Bash
sudo docker build -t rps-generator .
```

2. Запуск игры
Используйте флаг ***-it***, чтобы иметь возможность вводить свои ходы в консоли:

```Bash
sudo docker run -it rps-generator
```


**Исследование внутренностей**

Поскольку матрешка собирается внутри изолированного образа, промежуточные файлы не видны. Чтобы убедиться, что генерация прошла успешно, вы можете войти "внутрь" матрешки.

- Просмотр сгенерированных файлов
Запустите контейнер в режиме интерактивной оболочки:

```Bash
sudo docker run -it rps-generator bash
```

Внутри введите ***ls -l***. Вы увидите все слои системы:

- step3.py — Исходный скрипт.

- step2.sh — Создан уровнем Python.

- generator.cpp — Создан уровнем Bash.

- generator — Бинарный файл (результат компиляции C++).

- rsp.sh — Финальный результат.

Чтобы вернуться из контейнера в свою основную систему, введите:

```Bash
exit
```


**Логика игры**

Финальный скрипт rsp.sh — это не просто игра. При каждом запуске бот рассчитывает cheat_chance (шанс на жульничество) от 42% до 67%.

Если боту "везет", он подсматривает ваш выбор и выкидывает победный вариант.

Если нет — играет честно, выбирая случайный ход.


**Очистка**

Чтобы удалить созданный образ из системы после проверки:

```Bash
sudo docker rmi -f rps-generator
```
