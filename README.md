# Romashkovo Valley

Небольшое веб-приложение в формате мини-игры по мотивам мира Смешариков.  
Игрок управляет жизнью Ромашковой долины: каждый день ему предлагается событие и несколько вариантов решения, а выбор влияет на настроение деревни и состояние персонажей.

## Идея проекта

Цель проекта — сделать простую браузерную игру с интерактивным сюжетом, где пользователь не просто нажимает кнопки, а принимает решения и видит последствия своих действий.

В основе идеи:
- короткая игровая сессия на 7 дней;
- выбор из нескольких вариантов ответа;
- изменение параметров героев и общей атмосферы деревни;
- визуально приятный интерфейс с атмосферой сказочного мира.

## Функциональность

Сейчас приложение поддерживает:

- старт игры в браузере;
- отображение события текущего дня;
- выбор одного из трёх решений;
- обработку последствий выбора;
- обновление статистики персонажей;
- изменение настроения деревни;
- экран завершения игры с итоговыми результатами;
- перезапуск игры без перезагрузки логики проекта;
- оформление главной страницы с иллюстрацией Ромашковой долины.

## Технологии

В проекте используются:

- `Python` — серверная часть;
- `Flask` — веб-фреймворк;
- `C++` — игровая логика и обработка событий;
- `HTML / CSS / JavaScript` — клиентская часть;
- `Docker` — контейнеризация приложения;
- `Git / GitHub` — контроль версий и совместная работа.

## Предполагаемая архитектура

Проект разделён на несколько частей:

- `app.py`
  Flask-приложение. Отдаёт страницу, принимает запросы от фронтенда, вызывает C++-движок и хранит текущее состояние игры.

- `camomile.cpp`
  Основная игровая логика. Здесь описаны события, варианты выбора и изменения характеристик персонажей и деревни.

- `templates/index.html`
  Интерфейс игры. Отвечает за отображение событий, кнопки выбора, итоговый экран и визуальное оформление.

- `static/`
  Статические ресурсы, например изображения для интерфейса.

### Схема работы

1. Пользователь открывает главную страницу.
2. Flask отдаёт HTML-шаблон.
3. JavaScript запрашивает событие текущего дня.
4. Flask передаёт состояние игры в C++-программу.
5. C++ возвращает текст события или результат выбора в формате JSON.
6. Интерфейс обновляет текст, статистику и настроение деревни.

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/gislowloves/introduction-to-the-specialty.git
cd introduction-to-the-specialty
```

### 2. Собрать образ

```bash
docker build -t romashkovo .
```

### 3. Запустить контейнер

```bash
docker run -p 5000:5000 romashkovo
```

Открой в браузере: `http://localhost:5000`
